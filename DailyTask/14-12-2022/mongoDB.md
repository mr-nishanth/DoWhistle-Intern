### Working of MongoDB


- MongoDB Data stored in BSON format

- Written in C++

- NoSql does not use tables with rows and column



- DB(SQL) == DB(NoSql)

- Table(SQL) == Collection(NoSql)

- Column(SQL) == Field(NoSql)

- row(SQL) == document(NoSql)

- Table Join(SQL) == Embedded Document
